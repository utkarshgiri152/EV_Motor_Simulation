# EV_Motor_Simulation
# 🚗 EV Motor Speed Control using PID (Simulink)

## Overview
This project implements a closed-loop speed control system for a DC motor using a PID controller in MATLAB/Simulink. The goal is to maintain desired speed under different conditions and improve system performance.

## Objectives
- Model a DC motor system  
- Analyze open-loop response  
- Design a PID controller for speed control  
- Compare open-loop and closed-loop performance  
- Test system under load disturbance  

## Tools Used
- MATLAB  
- Simulink  

## Methodology
- Developed DC motor model using standard equations  
- Applied step input to observe open-loop behavior  
- Implemented feedback loop with PID controller  
- Tuned PID parameters for better response  
- Added load disturbance to test stability  

## Results
- Reduced overshoot  
- Improved settling time  
- Stable speed response achieved  
- System handled disturbances effectively  

## How to Run
1. Open MATLAB  
2. Open the file `ev_motor_control.slx`  
3. Run the simulation  
4. Observe outputs in Scope
